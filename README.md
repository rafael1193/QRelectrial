QRelectrial
===========

Español
-------

QRelectrial es un programa creado para leer la información de los carnés de socio de la Asociación Electrial, asociación de alumnos de Ingeniería Electrónica Industrial de la Universidad de Granada.

Para saber más sobre Electrial visita [nuestra web](http://aselectrial.blogspot.com).

***Datos técnicos***

Este Programa necesita .NET Framework 3.5 y ha sido probado en M$ Windows 7.

English
------

QRelectrial is a software for reading Electrial Association member cards, Granada university electronic engineering student association.

Learn more about Electrial at [our website](http://aselectrial.blogspot.com) (in spanish).

***Technical data***

This program needs .NET Framework 3.5 and it's proved to work in M$ Windows 7
